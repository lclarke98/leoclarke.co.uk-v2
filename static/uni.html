    Internet of Things coursework

    For my IoT coursdework I developed a room temperature and humidity monitoring system. I used two DHT22 temperature and humidity sensors, each connected to an esp8266 microcontroller. These were placed in two different locations in a room and send the data on a time interval to the google IoT core. From there a cloud function would insert the data into a MySQL database. Before the data was displayed on the web interface it was filterd trough a kalman filter which fused the readings from the two sensord to give more accurate readings.
